# Gunnel Habitat Dashboard

This project presents an interactive Shiny dashboard to analyze environmental factors affecting Gunnel presence.

## Features
- Time-series analysis
- Habitat comparison
- Scatter plots
- Interactive filters

## Tools Used
- R Shiny
- ggplot2
- dplyr

## How to Run
1. Open RStudio
2. Run app.R

## Dataset
Gunnels.csv
